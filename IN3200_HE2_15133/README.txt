Compilation and run of serial code:

gcc serial_main.c -o serial_main
./serial_main

Compilation and run of MPI code:

mpicc mpi_main.c -o mpi_main
mpiexec -n <number_of_processors> ./mpi_main

The example list used in the main programs are the list given in the task. Works for any other
list generated by random or typed in manually
