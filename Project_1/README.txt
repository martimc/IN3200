Compilation of the functions:

To compile and run the functions using table2D format:

- unparallized: make main1

- parallized: make main1_OMP

To compile and run the functions using CRS format:

- unparallized: make main2

- parallized: make main2_OMP

To compile and run the test function which times the functions:

- unparallized: make test

- parallized: make test_OMP
